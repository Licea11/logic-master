## Umbrales Críticos

### Detección
- `𝔽(x,t) < -θ` → ALERTA
- `𝔸ᵣₑₐₗ(x) > 0.7` → AUTÉNTICO
- `𝕀(x,t) > ℍ_crítico` → HIPOCRESÍA
- `𝔏(x) > 0.6` → LEGÍTIMO

### Inmunización
- `||ℑ⃗(y,t)|| > ℑ_min` → INMUNE
- `𝔸(y,t) > 𝔸_crítico` → ANCLADO
- `ℜ(y,x) > ℜ_min` → RESISTENTE

### Alerta
- `🚩ₛ(x,t) > 0.7` → CÓDIGO_ROJO
- `𝔾ₐc(x,T) > umbral_grad` → GRADUALISMO
- `ℛₜ(x,t) > 0.8` → PELIGRO_EXTREMO

### Recuperación
- `𝔖(y,τ) > 𝔖_mín` → SANIDAD_SUFICIENTE
- `𝒫ᵣₑₛₜ(y,t) > 0.9` → RESTAURADO
- `ℑ(y,t) ≥ 0.85` → INMUNIDAD_PERMANENTE

### Sistema Total
- `𝕊ₘ(y,x,H₃,t) > 0.9` → BLINDAJE_COMPLETO

## Ecuación Maestra del Sistema

```
𝕊ₘ(y,x,H₃,t) = [𝔻(t) + ℑ(t) + ℛ(t) + 𝔸ₜ(t)] / 4
```

Donde:
- `𝔻(t)` = sistemas de detección (Sistema I)
- `ℑ(t)` = sistemas de inmunización (Sistema II)
- `ℛ(t)` = sistemas de contraataque/recuperación (Sistemas III, V)
- `𝔸ₜ(t)` = sistemas de alerta temprana (Sistemas IV, VI)

**Objetivo**: lim[t→∞] 𝕊ₘ(y,t) = 1

**Condición de Victoria**:
```
∀x ∈ Falsos, ∀H₃ ∈ Espíritus_error:
  𝕊ₘ(y,x,H₃,t) > 0.9 ⟹ [𝒱(y,x,t) ≈ 0 ∧ ℋ(y,H₃,t) ≈ 0]
```
